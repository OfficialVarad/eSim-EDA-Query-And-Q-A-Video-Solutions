## About

This repository contains **video-based solutions to common queries related to eSim EDA**.  
Each solution is organized into folders based on the topic or issue.

The purpose of this repository is to make it easier for users to **find explanations and troubleshooting guides** through short demonstration videos.

---

## How to Use This Repository

1. Browse the folders listed in this repository.
2. Each folder represents a **specific topic or query**.
3. Open the folder to view the **README or files containing the video solution**.
4. Click the **YouTube link** provided to watch the explanation.

---

## Topics Covered

Topics included in this repository:
1. How to Create a Contributor account on eSim Portal 
2. How to Create a first project and draw a first schematic in eSim 2.5
3. How to Create a Subcircuit schematic in eSim 2.5
4. How to Create a Symbol for schematic in eSim 2.5
5. How to Create a Test circuit to test the developed subcircuit/IC in eSim 2.5

New query solutions will be added over time.
